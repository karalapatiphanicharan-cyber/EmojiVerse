# 🎨 EmojiVerse

> Turn words, images & imagination into living emoji art.

EmojiVerse is a creative emoji art studio designed with a focus on tactile interaction and physical feedback. This project transforms the digital emoji experience into a physical-feeling workstation using skeuomorphic design principles.

![EmojiVerse Logo](https://img.shields.io/badge/Emoji-Verse-FF6B6B?style=for-the-badge&logo=appveyor)

## 🚀 Project Overview

EmojiVerse allows users to create complex artwork using only emojis. Whether it's painting with emoji brushes, animating creations with a high-fidelity motion engine, or converting images into emoji mosaics.

## 🛠 Tech Stack

- **Frontend:** React, Vite
- **Styling:** Tailwind CSS (Custom Skeuomorphic System)
- **Animation:** Framer Motion (Presets: Rain, Explosion, Wave, etc.)
- **Export:** GIF.js & HTML2Canvas (Animation & Static exports)
- **Icons:** Lucide React

## ✨ Features (Phases 1-3 Complete)

- **3D Skeuomorphic UI:** Realistic surfaces, deep shadows, and tactile buttons.
- **Emoji Text Painter:** Convert text to emoji art using a 7x5 pixel matrix engine.
- **Advanced Painter:** Manual drawing canvas (50x30) with Brush, Fill, and Undo/Redo tools.
- **Multi-Emoji & Gradients:** Apply horizontal, vertical, and random gradients to your text art.
- **Animation Studio:** Full motion engine with presets, live timeline, and GIF export capability.
- **Physical Gallery:** Save your creations locally and preview them with animated thumbnails.

## ⚙️ Setup Instructions

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Build for production: `npm run build`
